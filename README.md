# frontend-web

React 19 + TypeScript + Vite enterprise CRM frontend.

## Tech Stack

| Concern | Library |
|---|---|
| Framework | React 19 |
| Language | TypeScript 6 |
| Build | Vite 8 |
| Styling | Tailwind CSS v4 |
| Routing | React Router v7 |
| Server state | TanStack Query v5 |
| Table | TanStack Table v8 |
| HTTP | Axios |
| UI | Radix UI + shadcn-style components |

## Getting Started

```bash
npm install
npm run dev
npm run build
npm run lint
```
